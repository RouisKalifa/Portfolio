Description du projet

Ce projet consiste en la mise en place d’un serveur web basé sur Nginx et déployé dans un conteneur Docker. Il permet d’afficher une page web en local et d’explorer les concepts de virtualisation et de gestion de conteneurs.


Objectifs :

  • Configurer un serveur web avec Nginx.
  • Déployer et gérer l’application via Docker.
  • Se familiariser avec les concepts de virtualisation et de conteneurs.
  • Explorer les notions liées aux données et aux outils associés.
  • Obtenir une certification Python pour valider les compétences en programmation.
