Projet de Site Web - Info Dev

Présentation du projet

Nous avons développé un site web dynamique en PHP avec une base de données MySQL pour l'entreprise Info-Dev. Ce site présente l’entreprise, ses services (réseaux et développement), ses projets, ses mentions légales, sa politique de confidentialité et ses contacts.


Design et Charte Graphique

Le site suit une charte graphique en bleu, noir et blanc, reflétant l’univers informatique. Un logo spécifique a été conçu et respecté tout au long du projet.


Maquettage et Arborescence

Deux maquettes ont été créées, la première réalisée sur Wireframe, puis une seconde adaptée avec Bootstrap. L’arborescence du site a été élaborée sur Draw.io pour structurer l’information.


Technologies utilisées

  • HTML/CSS, JavaScript pour l'affichage et l’interactivité
  • PHP/MySQL pour la gestion des données
  • Bootstrap pour l’adaptation mobile

Référencement SEO

Un travail de référencement a été effectué en intégrant des mots-clés stratégiques dans les balises <meta name="keywords"> afin d’optimiser la visibilité du site sur les moteurs de recherche.


Base de données et Formulaires

Un formulaire en JavaScript permet la collecte de données des visiteurs. La connexion avec la base de données se fait via PDO et est gérée avec PhpMyAdmin.



Ce projet a permis de mettre en place un site web complet, interactif et responsive répondant aux besoins d'Info-Dev.
